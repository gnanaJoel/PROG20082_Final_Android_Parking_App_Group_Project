Park'in Go App Manual

Park'in Go application helps the user to save time by finding a parking spot, pay for the time slot and can able to access google maps to find parking spots at the current location.

This User Manual will help the user to understand the Application and guide how to use it.

SignIn:

The app first launches the Sign In window to let the user to Sign In if they are already registered.

Sign Up:

If the user, not a member and want to sign up, click on the sign-up option, it will navigate to the Signup page and ask's user information and details.

Once the user sign up and makes an account, they will be redirected back to the sign-in page and will be able to Log in.

Update Information:

If the user wants to make changes to their account they can click on the menu on the top right corner and select update.

This will open a dialog box to make changes to their accounts.

Once the user fills the form, he can click on the update button to make changes or if they wish not to make changes to their account, then they can cancel the form.

Add Parking/View Receipts:

When the user clicks on the navigation drawer on the top left, they will see Add Parking and View Receipts on the bottom.

When the user selects on add parking option, it will direct them to the parking page and can book the timings to park.

Enter the Building code to know the locations, time chosen to park and the car details. Then the user can click Park my car to get the online receipt.

Communicate:

If the user has any technical difficulties or needs assistance, they can use our inbuilt CALL OR EMAIL option. The user has two choices,

By clicking the 'Call' option, the user will be directed to the call page where they can get the customer assistance number or they can simply click on the button to make the call.

Or by clicking the 'Email' option, the user will be directed to the email page where they can get the customer complaint form to fill and send it.
